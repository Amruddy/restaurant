# Restaurant Website Spec

## Цель

Создать современный сайт ресторана, который помогает гостю быстро понять атмосферу, посмотреть меню, узнать часы работы и забронировать стол.

## Основные разделы

- Первый экран: название ресторана, короткое позиционирование, кнопка бронирования.
- Меню: сезонные блюда, цены, краткие описания.
- Пространство: зал, приватная комната, открытая кухня.
- События: дегустации, ужины, винные вечера.
- Бронирование: форма заявки.
- Контакты: адрес, телефон, email, часы работы.

## Тон

Русский язык, спокойный ресторанный стиль, без рекламной перегрузки.

## Технический старт

Проект начинается как статический сайт:

- HTML
- CSS
- JavaScript без сборки

Фреймворк можно добавить позже, если появятся реальные причины: много страниц, CMS, сложное состояние, интеграции или админка.

# Sample Menu

## Назначение

Это тестовое меню для разработки MVP. Оно нужно, чтобы проектировать меню, корзину, доставку и визуальные карточки до появления финального ресторанного меню.

Эти позиции не являются утвержденным финальным меню ресторана. Перед публикацией их нужно заменить или подтвердить.

## Использование

Для разработки эти позиции можно перенести в `data/menu.ts`.

Все блюда считаются доступными для доставки, пока в `docs/feature-specs/delivery.md` не появятся ограничения.
